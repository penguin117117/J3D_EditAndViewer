# J3D_EditAndViewer
このプロジェクトはBDLファイルの知識を得る目的で作成されています。</br>
現在、編集機能はありません。</br>
閲覧機能はSHP1セクションのReNameMapまで読み込めます。</br>
現段階では、モデルの表示機能はありません。

## 読み込み可能
- [x] INF1
- [x] VTX1
- [x] EVP1
- [x] DRW1
- [x] JNT1
- [ ] SHP1
- [ ] MAT3
- [ ] MDL3
- [ ] TEX1

## 開発環境
<b>Visual Studio 2019</b></br>
<b>.NET Framework 4.8</b></br>
